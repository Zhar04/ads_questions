# Картинки к вопросам

Сюда кладутся иллюстрации (диаграммы, блок-схемы, графы) к вопросам.

Имя файла должно **точно** совпадать с полем `image` у вопроса в банке.

## Текущие нужные файлы (банк АиСД)

| Файл | Вопрос | Что должно быть на картинке |
|------|--------|------------------------------|
| `ads-026.png` | ADS-026 | взвешенный ориентированный граф |
| `ads-053.png` | ADS-053 | разрешение коллизий в хеш-таблице |
| `ads-113.png` | ADS-113 | блок-схема оператора |

Формат — **PNG** (имя строчными буквами, ровно как в таблице). Положите файл сюда — он
автоматически появится в вопросе на сайте.
